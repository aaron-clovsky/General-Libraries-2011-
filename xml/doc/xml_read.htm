NAME

xml_read - Read an XML file into an xml structure

 SYNOPSIS

#include "xml.h"
int xml_read(xml *document, const char *file_name);

 DESCRIPTION

The xml_read() function creates an xml document in the xml structure pointed to by the document argument from the file indicated in the file_name argument.
This document is available as a null terminated string in the data member of the xml strucure.
xml_read() should not be called with a pointer to an xml structure that has already been initialized by a previous call to xml_read(), xml_create() or xml_import().

 RETURN VALUE

Upon successful completion the function returns zero.

 ERRORS

In case of an error one of the following values will be returned:
[XML_ERROR_NULL_POINTER_ERROR]
The document or file_name arguments are NULL.
[XML_ERROR_FILE_IO_ERROR]
An error occured in a call to fopen() or fread().
This usually indicates that the file file_name does not exist.
[XML_ERROR_MALLOC_ERROR]
A call to malloc() has failed due to insufficient memory.

 EXAMPLES

xml document;

xml_read(&document, "file.xml");
