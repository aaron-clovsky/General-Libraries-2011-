NAME

id3_free - Free memory associated with an id3 tag

 SYNOPSIS

#include "id3.h"
void id3_free(id3 info);

 DESCRIPTION

The id3_free() function frees memory related to the id3 structure pointed to by the info argument.
id3_free() should only be called after an id3 structure has been initialized by a call to id3_read().

 RETURN VALUE

none

 ERRORS

No errors are handled

 EXAMPLES

id3 tag;

if (!id3_read("some_mp3.mp3", &tag, 0))
{
	printf("mp3 title: %s\n", tag.title);
	
	id3_free(tag);
}
else
{
	printf("Error reading tag\n");
}
