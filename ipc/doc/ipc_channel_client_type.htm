NAME

ipc_channel_client_type - Determine the client type of an inter-process communication client

 SYNOPSIS

#include "ipc.h"
int ipc_channel_client_type(ipc_channel_server_client client);

 DESCRIPTION

The ipc_channel_client_type() function determines the client type of an inter-process communication client.
The client argument is an ipc_channel_server_client type variable the value of which was returned by a previous call to ipc_channel_receive().

 RETURN VALUE

[IPC_CHANNEL_HALF_DUPLEX]
The client is a half-duplex client.
[IPC_CHANNEL_FULL_DUPLEX]
The client is a full-duplex client.

 ERRORS

No errors are handled.

 EXAMPLES

See the documentation for ipc_channel_create() for an example of ipc_channel_client_type().
