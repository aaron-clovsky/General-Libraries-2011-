NAME

thread_tls_destroy - Free memory associated with a thread local pointer

 SYNOPSIS

#include "thread.h"
void thread_tls_destroy(thread_tls tls);

 DESCRIPTION

The thread_tls_destroy() function frees memory associated with a thread local pointer.

The function has no effect if the address of the thread_tls variable specified by the tls argument was not previously initialized with a call to thread_tls_create().

 RETURN VALUE

None.

 ERRORS

No errors are handled.

 EXAMPLES

See the documentation for thread_tls_create() for an example of thread_tls_destroy()
