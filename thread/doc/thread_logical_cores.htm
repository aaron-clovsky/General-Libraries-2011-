NAME

thread_logical_cores - Determine the number of logical cores on the current system.

 SYNOPSIS

#include "thread.h"
int thread_logical_cores(void);

 DESCRIPTION

The thread_logical_cores() function determines the number of logical cores on the current system.
This is useful as it is also the number of concurrent hardware supported threads the system is capable of running.

 RETURN VALUE

The number of logical cores on the current system.

 ERRORS

No errors are handled.

 EXAMPLES

printf("Hardware threads available: %d\n", thread_logical_cores());
