NAME

thread_mutex_unlock - Unlock a mutex

 SYNOPSIS

#include "thread.h"
int thread_mutex_unlock(thread_mutex *mutex);

 DESCRIPTION

The thread_mutex_unlock() function unlocks a mutex pointed to by the mutex argument.
The mutex argument must point to a thread_mutex type variable which has been initialized to THREAD_MUTEX_INITIALIZER and locked with thread_mutex_lock() or thread_mutex_trylock().

 RETURN VALUE

Upon successful completion the function returns zero.

 ERRORS

In case of an error one of the following values will be returned:
[THREAD_ERROR_NULL_POINTER_ERROR]
The mutex argument is NULL.
[THREAD_ERROR_SYSTEM_CALL_ERROR]
A call to an operating system function failed.

 EXAMPLES

See the documentation for thread_mutex_lock() for an example of thread_tls_unlock()
