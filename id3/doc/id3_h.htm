NAME

id3.h - MP3 ID3 tag reading library

 SYNOPSIS

#include "id3.h"

COMPATIBILITY

ANSI C

 DESCRIPTION

The id3.h header file provides ID3 tag reading and ID3 tag memory management functions.

This header defines the type id3 with the following definition: 

typedef struct 
{
 	char *artist;
	char *album;
	char *title;
	char *track;
	char *year;
	void *image;
	int   image_size;
	int   image_type;
	int   encoding;
	int   version;
} id3;
The following functions are defined: 
int  id3_read(const char *file_name, id3 *info, int flags);
void id3_free(id3 info);
