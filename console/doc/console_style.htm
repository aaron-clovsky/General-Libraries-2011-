NAME

console_style - Change console text style

 SYNOPSIS

#include "console.h"
void console_style(int style);

 DESCRIPTION

The console_style() function changes the style of text subsequently printed to the console.
The style argument specifies the style to be applied.
This style is constructed from a binary combination of the following constants:

Foreground Colors 

	CONSOLE_FG_COLOR_BLACK
	CONSOLE_FG_COLOR_RED
	CONSOLE_FG_COLOR_GREEN
	CONSOLE_FG_COLOR_YELLOW
	CONSOLE_FG_COLOR_BLUE
	CONSOLE_FG_COLOR_MAGENTA
	CONSOLE_FG_COLOR_CYAN
	CONSOLE_FG_COLOR_WHITE
Background Colors 
	CONSOLE_BG_COLOR_BLACK
	CONSOLE_BG_COLOR_RED
	CONSOLE_BG_COLOR_GREEN
	CONSOLE_BG_COLOR_YELLOW
	CONSOLE_BG_COLOR_BLUE
	CONSOLE_BG_COLOR_MAGENTA
	CONSOLE_BG_COLOR_CYAN
	CONSOLE_BG_COLOR_WHITE
Text Styles 
	CONSOLE_STYLE_DEFAULT
	CONSOLE_STYLE_BOLD
Note: Console style will NOT return to default on process termination, call:
console_style(CONSOLE_STYLE_DEFAULT);
to restore the default style.

 RETURN VALUE

none

 ERRORS

No errors are handled

 EXAMPLES

printf("Normal\n");

console_style(CONSOLE_COLOR_FG_BLACK | CONSOLE_COLOR_BG_WHITE);

printf("Black on White\n");

console_style(CONSOLE_COLOR_FG_GREEN | CONSOLE_COLOR_BG_BLACK | CONSOLE_STYLE_BOLD);

printf("Bold Green on Black\n");

console_style(CONSOLE_STYLE_DEFAULT);
