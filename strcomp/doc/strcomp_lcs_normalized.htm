NAME

strcomp_lcs_normalized - Compute the similarity of two strings using the longest common substring

 SYNOPSIS

#include "strcomp.h"
float strcomp_lcs_normalized(const char *str1, const char *str2);

 DESCRIPTION

The strcomp_lcs() function computes the longest common substring of two given strings and divides this value by the length of the longer string, normalizing the value to a range of [0..1].
Character comparison in strcomp_lcs_normalized() ignores case.

 RETURN VALUE

A normalized value inidicating the percentage of str1 and str2 that are identical.

 ERRORS

In case of an error one of the following values will be returned:
[STRCOMP_ERROR]
The str1 or str2 arguments are NULL.

 EXAMPLES


strcomp_lcs_normalized("Hello World", "World"); //Returns 0.4545
