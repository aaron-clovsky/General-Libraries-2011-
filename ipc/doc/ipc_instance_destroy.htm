NAME

ipc_instance_destroy - Destroy a unique named instance

 SYNOPSIS

#include "ipc.h"
void ipc_instance_destroy(ipc_instance *instance);

 DESCRIPTION

The ipc_instance_destroy() function destroys a unique named instance previously created with ipc_instance_create(). Once the instance is destroyed it can be successfully created again by other proccesses on the system.
The instance argument is a pointer to an ipc_instance type variable, the address of this variable must have been previously passed to ipc_instance_create().

 RETURN VALUE

None.

 ERRORS

No errors are handled.

 EXAMPLES

See the documentation for ipc_instance_create() for an example of ipc_instance_destroy().
