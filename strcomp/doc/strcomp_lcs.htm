NAME

strcomp_lcs - Compute longest common substring of two strings

 SYNOPSIS

#include "strcomp.h"
int strcomp_lcs(const char *str1, const char *str2);

 DESCRIPTION

The strcomp_lcs() function computes the longest common substring of two given strings.
Character comparison in strcomp_lcs() ignores case.

 RETURN VALUE

A positive integer indicating the length of the longest common substring of str1 and str2.

 ERRORS

In case of an error one of the following values will be returned:
[STRCOMP_ERROR]
The str1 or str2 arguments are NULL.

 EXAMPLES


strcomp_lcs("Hello World", "World"); //Returns 5
