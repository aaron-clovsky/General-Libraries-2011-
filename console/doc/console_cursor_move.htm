NAME

console_cursor_move - Move cursor by a given offset

 SYNOPSIS

#include "console.h"
void console_cursor_move(int x, int y);

 DESCRIPTION

The console_cursor_move() function moves the console cursor by a given offset.

 RETURN VALUE

none

 ERRORS

No errors are handled

 EXAMPLES

console_clear();

console_cursor_move(2, 2);

printf("This will start printing at (2, 2)");

console_cursor_move(10, 10);

printf("This will start printing at (12, 12)");
