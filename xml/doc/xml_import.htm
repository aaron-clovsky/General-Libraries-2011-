NAME

xml_import - Copy a null terminated string into an xml structure

 SYNOPSIS

#include "xml.h"
int xml_import(xml *document, const char *data)

 DESCRIPTION

The xml_import() function takes a null terminated string and creates an xml structure from it.
The document argument specifies the xml structure to write to.
The data argument is a null terminated string containing an XML document.
xml_import() should not be called with a pointer to an xml structure that has already been initialized by a previous call to xml_import(), xml_read() or xml_create().

 RETURN VALUE

Upon successful completion the function returns zero.

 ERRORS

In case of an error one of the following values will be returned:
[XML_ERROR_NULL_POINTER_ERROR]
The document or data arguments are NULL.
[XML_ERROR_MALLOC_ERROR]
A call to malloc() has failed due to insufficient memory.

 EXAMPLES

xml   document;
const char *my_xml = "<tag>I love XML</tag>";
