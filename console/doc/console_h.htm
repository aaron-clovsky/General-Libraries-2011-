NAME

console.h - Console manipulation library

 SYNOPSIS

#include "console.h"

COMPATIBILITY

Win32/ANSI compatible (UNIX) console

 DESCRIPTION

The console.h header file provides console cursor and text style control functions.
The following functions are defined: 
void console_clear(void);
void console_cursor_goto(unsigned int x, unsigned int y);
void console_cursor_move(int x, int y);
void console_style(int style);

NOTES

The top left character displayed in the console is considered to be position (1, 1).

The behavior of this library in a threaded environment differs between operating systems and compilers.
Under UNIX environments if the compiler uses a thread-safe implementation of printf() then this library is thread-safe.
Under WIN32 this library is NOT thread-safe, to mitigate issues which may occur in this situation only manipulate the console style and cursor from a single thread.
