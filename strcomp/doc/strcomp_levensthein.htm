NAME

strcomp_levensthein - Compute the levensthein edit distance between two strings

 SYNOPSIS

#include "strcomp.h"
int strcomp_levensthein(const char *str1, const char *str2);

 DESCRIPTION

The strcomp_levensthein() computes the levensthein edit distance between two strings.
Character comparison in strcomp_levensthein() ignores case.

 RETURN VALUE

The levensthein edit distance of str1 and str2.

 ERRORS

In case of an error one of the following values will be returned:
[STRCOMP_ERROR]
The str1 or str2 arguments are NULL.
OR
STRCOMP_FIXED_STORAGE is non-zero and the storage requirements exceed STRCOMP_FIXED_STORAGE bytes.

 EXAMPLES

strcomp_levensthein("Hello World", "Hi World"); //Returns 4
