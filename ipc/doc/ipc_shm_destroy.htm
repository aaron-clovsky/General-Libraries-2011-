NAME

ipc_shm_destroy - Destroy a named shared memory region

 SYNOPSIS

#include "ipc.h"
void ipc_shm_destroy(ipc_shm *shm);

 DESCRIPTION

The ipc_shm_destroy() function destroys a named shared memory region created by ipc_shm_create(). Once ipc_shm_destroy() has been called any process can create a shared memory region with the same name as the destroyed region.
The shm argument is a pointer to an ipc_shm type variable which has been initialized by a previous call to ipc_shm_create().

 RETURN VALUE

None.

 ERRORS

No errors are handled.

 EXAMPLES

See the documentation for ipc_shm_create() for an example of ipc_shm_destroy().
