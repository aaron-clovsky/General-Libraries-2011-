NAME

strcomp_levensthein_normalized - Compute the normalized similarity of two strings using the levensthein edit distance

 SYNOPSIS

#include "strcomp.h"
float strcomp_levensthein_normalized(const char *str1, const char *str2);

 DESCRIPTION

The strcomp_levensthein_normalized() computes the levensthein edit distance between two strings and normalizes the result to [0..1] using the formula: 
		1 - (Distance / N)
Where Distance is the levensthein edit distance and N is the length of the longer string.
Character comparison in strcomp_levensthein_normalized() ignores case.

 RETURN VALUE

The normalized similarity of str1 and str2 computed from the levensthein edit distance.

 ERRORS

In case of an error one of the following values will be returned:
[STRCOMP_ERROR]
The str1 or str2 arguments are NULL.
OR
STRCOMP_FIXED_STORAGE is non-zero and the storage requirements exceed STRCOMP_FIXED_STORAGE bytes.

 EXAMPLES

strcomp_levensthein("Hello World", "Hi World"); //Returns 0.6363
