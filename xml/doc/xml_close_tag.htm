NAME

xml_close_tag - Close an XML tag

 SYNOPSIS

#include "xml.h"
int xml_close_tag(xml *document, const char *tag);

 DESCRIPTION

The xml_close_tag() function closes a tag for an XML element (ie </tag>).
The document argument specifies the xml document to append the close tag to.
The tag argument specifies the tag name to use.

 RETURN VALUE

Upon successful completion the function returns zero.

 ERRORS

In case of an error one of the following values will be returned:
[XML_ERROR_NULL_POINTER_ERROR]
The document or tag arguments are NULL.
[XML_ERROR_INITIALIZATION_ERROR]
The xml structure has not been initalized with the xml_create(), xml_read() or xml_import() functions.
[XML_ERROR_FILE_IO_ERROR]
An error occured in a call to fopen() or fwrite().

 EXAMPLES

xml document;

xml_create(&document);

xml_open_tag(&document, "tag&attributeOne&attributeTwo", "value", "another value");

xml_close_tag(&document, "tag");
