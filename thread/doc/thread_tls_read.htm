NAME

thread_tls_read - Retrieve the value of a thread local pointer

 SYNOPSIS

#include "thread.h"
int thread_tls_read(thread_tls tls, void **value);

 DESCRIPTION

The thread_tls_read() function retrieves the value of a thread local pointer indicated by the tls argument.

The address of the tls argument must have previously been previously initialized with a call to thread_tls_create()

The value argument indicates the address of a void pointer to place the current value of the thread local pointer into.

 RETURN VALUE

Upon successful completion the function returns zero.

 ERRORS

In case of an error one of the following values will be returned:
[THREAD_ERROR_NULL_POINTER_ERROR]
The value argument is NULL.
[THREAD_ERROR_SYSTEM_CALL_ERROR]
A call to an operating system function failed.

 EXAMPLES

See the documentation for thread_tls_create() for an example of thread_tls_read()
