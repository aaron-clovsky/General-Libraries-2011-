NAME

thread_semaphore_destroy - Free memory associated with an unnamed semaphore

 SYNOPSIS

#include "thread.h"
void thread_semaphore_destroy(thread_semaphore *semaphore);

 DESCRIPTION

The thread_semaphore_destroy() function frees memory associated with an unnamed semaphore.

The function has no effect if the address of the thread_semaphore variable specified by the semaphore argument was not previously initialized with a call to thread_semaphore_create().

 RETURN VALUE

None.

 ERRORS

No errors are handled.

 EXAMPLES

See the documentation for thread_semaphore_create() for an example of thread_semaphore_destroy()
