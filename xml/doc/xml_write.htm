NAME

xml_write - Write an XML file from an xml structure

 SYNOPSIS

#include "xml.h"
int xml_write(xml document, const char *file_name);

 DESCRIPTION

The xml_write() function writes an xml document from the xml structure passed in the document argument to the file indicated in the file_name argument.

 RETURN VALUE

Upon successful completion the function returns zero.

 ERRORS

In case of an error one of the following values will be returned:
[XML_ERROR_NULL_POINTER_ERROR]
The file_name argument is NULL.
[XML_ERROR_FILE_IO_ERROR]
An error occured in a call to fopen() or fwrite().

 EXAMPLES

xml document;

xml_read(&document, "file.xml");

xml_write(document, "copy.xml");
