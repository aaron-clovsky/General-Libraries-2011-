NAME

console_clear - Clear the console

 SYNOPSIS

#include "console.h"
void console_clear(void);

 DESCRIPTION

The console_clear() function clears the console and moves the cursor to the position (1, 1).

 RETURN VALUE

none

 ERRORS

No errors are handled

 EXAMPLES

printf("This will not be seen\r\n");

console_clear();

printf("This will be seen\r\n");
