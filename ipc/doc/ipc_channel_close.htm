NAME

ipc_channel_close - Close a connection to an inter-process commmunication channel

 SYNOPSIS

#include "ipc.h"
void ipc_channel_close(ipc_channel_client *channel);

 DESCRIPTION

The ipc_channel_close() function closes a connection to an inter-process communication (IPC) channel opened by ipc_channel_open().
The channel argument is a pointer to an ipc_channel_client type variable which has been initialized by a previous call to ipc_channel_open().

 RETURN VALUE

None.

 ERRORS

No errors are handled.

 EXAMPLES

See the documentation for ipc_channel_create() for an example of ipc_channel_close().
