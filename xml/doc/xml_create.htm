NAME

xml_create - Create an XML document

 SYNOPSIS

#include "xml.h"
int xml_create(xml *document);

 DESCRIPTION

The xml_create() function creates an xml document in the xml structure pointed to by the document argument. This document is available as a null terminated string upon creation in the data member of the xml strucure.
xml_create() should not be called with a pointer to an xml structure that has already been initialized by a previous call to xml_create(), xml_read() or xml_import().

 RETURN VALUE

Upon successful completion the function returns zero.

 ERRORS

In case of an error one of the following values will be returned:
[XML_ERROR_NULL_POINTER_ERROR]
The document argument is NULL.
[XML_ERROR_MALLOC_ERROR]
A call to malloc() has failed due to insufficient memory.

 EXAMPLES

xml document;

xml_create(&document);
