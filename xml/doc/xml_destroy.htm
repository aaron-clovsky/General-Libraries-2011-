NAME

xml_destroy - Free memory associated with an XML document

 SYNOPSIS

#include "xml.h"
void xml_destroy(xml *document);

 DESCRIPTION

The xml_destroy() function frees memory related to the xml structure pointed to by the document argument.
xml_destroy() should be called after an xml structure is initialized by a call to xml_create(), xml_read() or xml_import().

 RETURN VALUE

none

 ERRORS

No errors are handled

 EXAMPLES

xml document;

xml_create(&document);

xml_destroy(&document);
