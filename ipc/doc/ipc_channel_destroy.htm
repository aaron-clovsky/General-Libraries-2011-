NAME

ipc_channel_destroy - Destroy an inter-process commmunication channel

 SYNOPSIS

#include "ipc.h"
void ipc_channel_destroy(ipc_channel_server *channel);

 DESCRIPTION

The ipc_channel_destroy() function destroys an inter-process communication (IPC) channel created by ipc_channel_create(). Once ipc_channel_destroy() has been called any process can create an IPC channel with the same name as the destroyed channel.
The channel argument is a pointer to an ipc_channel_server type variable which has been initialized by a previous call to ipc_channel_create().

 RETURN VALUE

None.

 ERRORS

No errors are handled.

 EXAMPLES

See the documentation for ipc_channel_create() for an example of ipc_channel_destroy().
